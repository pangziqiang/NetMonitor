# 流量统计页设计规范

## 原始设计文件

- `chart-playground.html` — 交互式图表调节器（可直接浏览器打开）
- `chart-design-params.json` — 设计参数 JSON（可1:1还原）
- `chart-render-spec.md` — 渲染规范文档

## 页面结构（三视图）

```
┌──────────────────────────────────────────────────────────────┐
│  标题栏（图标 + "流量统计"）                                     │
├──────────────────────────────────────────────────────────────┤
│  控制栏：日/周/年 切换  +  日期选择器（仅日视图）                  │
├──────────────────────────────────────────────────────────────┤
│  统计栏：总下载 | 总上传 | 峰值↓ | 峰值↑                        │
├──────────────────────────────────────────────────────────────┤
│  下载流量柱状图（24/24/24 根柱子，共享 Y 轴最大值）              │
├──────────────────────────────────────────────────────────────┤
│  上传流量柱状图（同上）                                          │
└──────────────────────────────────────────────────────────────┘
```

### 三种视图的数据与标签

| 视图 | 柱子数量 | 主标签 | 副标签 | 未来柱子判定 | 数据源 |
|------|----------|--------|--------|-------------|--------|
| **日** | 24（00:00~23:00） | `HH:00` | 空 | `hour > currentHour` | 分钟级聚合 + 实时补齐当前小时 |
| **周** | 24（从最早数据周一起） | `周一`~`周日` | `MM/DD` | `dateStr > todayStr` | 日级汇总（hourly 聚合） |
| **年** | 24（当年 1 月起） | `1月`~`12月` | `YYYY` | `monthKey > currentMonthKey` | 月级汇总（hourly 聚合） |

| cH | 200 | 图表高度 |
| statsH | 60 | 统计栏最小高度 |

| xS2 | 9 | 副标签字号 |

| xGap | 3 | 主副标签间距 |
| manFS | 12 | 数值字号 |
| vg | 2 | 数值距柱顶 |

## 各元素样式

### 统计栏 (stats)
- 布局：flex，gap: 20px
- 内边距：10px 14px
- 背景：rgba(255,255,255,0.03)
- 圆角：8px
- 标签：字号 10px，颜色 rgba(255,255,255,0.5)
- 数值：字号 16px，font-weight: 700，monospaced

### 图表容器 (chart)
- 背景：rgba(255,255,255,0.03)
- 圆角：8px
- 内边距：12px

### 图表名称 (chart-name)
- 字号：11px，font-weight: 600，颜色 #888

### 绘图区 (plot-area)
- 高度：200px
- 左边框：1px solid rgba(255,255,255,0.08)
- 底边框：1px solid rgba(255,255,255,0.08)

### 柱子 (bar-rect)
- 圆角：2px 2px 0 0（仅顶部）
- 未来柱子：高度固定 3px

### 数值标签 (bar-val)
- 字号：12px，font-weight: 600，monospaced
- 距柱顶：2px
- 所有非未来柱子都显示（包括0）

### X轴标签
- 主标签：12px，rgba(255,255,255,0.5)
- 副标签：9px，rgba(255,255,255,0.25)
- 间距：3px

## 数据

- **日视图**：24 根柱子（00:00~23:00），分钟级聚合 + 实时补齐当前小时
- **周视图**：24 根柱子（从最早数据所在周的周一开始），日级汇总
- **年视图**：24 根柱子（当年 1 月起），月级汇总
- 未来柱子：当前时间之后的显示为灰色 3px 细条
- 下载/上传共享 Y 轴最大值 `niceMax`
- 峰值时间记录在 hourly 表的 `peak_down_time` / `peak_up_time`
